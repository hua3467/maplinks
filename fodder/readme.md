# Rodder

## Libraries used

### SVG.js

<https://svgjs.com/docs/3.0/>

Include the script to the document head

```html
<!DOCTYPE html>
<html>
<head>
  <title>SVG.js</title>
  <script src="https://cdn.jsdelivr.net/npm/@svgdotjs/svg.js@3.0/dist/svg.min.js"></script>
</head>
<body>

</body>
</html>
```
